## Description

It is web app built using Laravel(5.8) framework and MySql database. Users can perform tasks such as Login, Register, Logout, Edit Profile.
### Dependencies 

- [Mews Captcha](https://github.com/mewebstudio/captcha)
- [Mottie Keyboard](http://mottie.github.io/Keyboard/)
- [BootStrap 4](https://getbootstrap.com/)
